# Bristol zoo monkeys — parody article

A standalone, self-contained webpage styled like a BristolLive news article. All CSS and JavaScript are inline, so it runs anywhere with no build step or dependencies.

## View it

Open `index.html` in any browser, or publish it with GitHub Pages.

## Publishing with GitHub Pages

1. Push this repo to GitHub.
2. Go to **Settings → Pages**.
3. Under **Source**, choose "Deploy from a branch", select `main` and `/ (root)`, and save.
4. The site goes live at `https://<username>.github.io/<repo>/` after a minute or two.

## Notes

This is a parody page. Comments, voting and the newsletter signup are client-side only and reset on refresh.
